# PiroZangi
札幌市厚別区の新さっぽろ地区を代表するカラオケピロスというお店をネタにしたザンギたたきゲームです。

STARTボタンを押したら、あとはひたすらザンギをたたいてください。
あ、ザンギ以外の人をたたくと減点になるので注意しましょう。
